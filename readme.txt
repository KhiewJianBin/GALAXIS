Singapore poly FYP Project

Galaxis - 1v1 shooter pvp battle, with power ups, tower defense. etc
Made targeting the OUYA platform, using xamarin Monogame

See Credits in Game


Controls:

Player 1

Movement: WASD
Scroll Shop Left: Q
Scroll Shop Right: E
Buy Shop Item: Tab
Activate Ability L-Shift

Player 2

Movement: Arrow Keys
Scroll Shop Left: Page Up
Scroll Shop Right: Page Down
Buy Shop Item: Enter
Activate Ability R-Shift
